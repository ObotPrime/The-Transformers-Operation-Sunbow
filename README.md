How to play: 
 - Arrow keys to move
 - Z to shoot
 - 3 Lasers can be shot at a time
 - Destroy all of the Decepticons to win
 - If you get hit, you die
 - Power-ups are available to give you upgrades
 - Enter the Konami Code for a surprise!
